# AirVent DePIN Dashboard (Gamified Edition)

[🚀 Live Demo](https://airvent-depin-dashboard.vercel.app)

블록체인 기반 공기질 데이터 보상 플랫폼, **AirVent**의 대시보드입니다.
사용자에게 채굴의 즐거움을 주는 **Gamification** 요소와 **AI 데이터 검증** 시뮬레이션이 포함되어 있습니다.

## 주요 기능 (Key Features)

### 1. Gamified Reward Mining
- **MiningCard**: 실시간으로 블록체인 블록이 생성되고 보상(AVP)이 채굴되는 과정을 시각화했습니다.
- 사용자는 대시보드를 켜두는 것만으로도 포인트가 쌓이는 시각적 피드백을 받습니다.

### 2. AI Data Verification (ChatGPT Simulation)
- **AiVerificationPanel**: 수집된 공기질 데이터를 AI 에이전트(Simulated GPT-5.2)가 실시간으로 분석합니다.
- 데이터 패턴에 따라 "환기 필요", "정상 범위" 등의 코멘트를 타이핑 효과와 함께 제공합니다.

### 3. Hyperlocal Air Quality Monitoring
- PM2.5, PM10, CO2, VOC 등 실시간 공기질 KPI를 한눈에 볼 수 있습니다. (Mock Data 기반)
